## 简介

这是一个基于 [FastAPI](https://fastapi.tiangolo.com/) 和 [Vue](https://cn.vuejs.org/) 的个人博客系统。

## 有啥不一样的？

这是我单纯想要做一个更加自由的博客系统，而不是被框架所限制。

## 为啥不用静态的？
